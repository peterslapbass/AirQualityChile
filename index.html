Cómo se calculan los datos
Índice ICA Chile
El Índice de Calidad del Aire (ICA) se calcula interpolando linealmente entre breakpoints del estándar EPA, adaptados para Chile. Cada contaminante tiene su propia escala.

Clasificación por colores
Cada estación en el mapa muestra un círculo cuyo color corresponde al valor ICA más alto entre todos los contaminantes medidos. El radio del círculo escala dinámicamente según la magnitud del ICA.
Campo de viento interpolado
Las estaciones meteorológicas de RedMeteo reportan velocidad y dirección del viento. Con esos datos se genera un grid interpolado de 50×50 celdas usando interpolación bilineal (SciPy). Las partículas animadas representan el flujo del viento en cada punto del mapa.
Normalización de contaminantes
La API del SINCA usa distintos nombres para un mismo contaminante (ej: "MP-2,5", "PM25", "PM2.5"). El sistema normaliza todos estos nombres a un formato estándar antes de procesarlos.

Alertas
Una estación genera alerta cuando cualquiera de sus contaminantes supera el umbral configurable (por defecto ICA > 100, equivalente a categoría "Mala"). Puedes ajustar este umbral desde el panel lateral.
Modo daltónico
